Results

We compared different text mining models with different datasets in our application to see how models behave. In the first approach we used the data about movies provided in the lab, and with this data we created one endpoint for each machine to generate the results. We sent as a parameter the name of the dataset through a GET request to let the api know what dataset we wanted to use, as results we received an accuracy of 47% for TFIDF, 40% for Ngrams and 42% for Bags of Words as expected. This results showed that our machine are working well. Next we tried experimenting with a different dataset that we found at www.kaggle.com/datasets. This dataset is about songs lyrics and as fields it contains name, lyrics, gender and author. With this dataset we trained a model that could predict the gender of a song based on the lyrics.
